## TodoList

Built a todo list with with Laravel.


## Installation

    
```bash
git clone https://github.com/Loydtafireyi/BootcampTodolist.git
cd BootcampTodolist
composer install
```

##### On Windows
    
    ```bash
    copy .env.example .env
    ```

##### On Mac

    ```bash
    cp .env.example .env
    ```

```bash
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```
